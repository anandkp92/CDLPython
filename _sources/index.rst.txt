.. CDL Python documentation master file

Welcome to CDL Python's documentation!
======================================

CDL Python is a Python implementation of the ASHRAE 231P Control Description Language (CDL),
providing a library of elementary control blocks for building automation and control systems.

This package includes:

* **cdl_python**: Core CDL block library with Reals, Integers, Logical, and other control blocks
* **cdl_translator**: Tools for translating CXF (CDL Exchange Format) to Python code
